look at the format of all the scripts under Scripts for inspiration.
Adjust the script 'non_pl_devices.py' in Scripts --> Alarms_And_Events --> Non_PacketLight_Devices --> PythonScripts, by this steps:

create side function called "verify_element_added(element_name)" that:
go to 'Management Map' page.
open navigation info via "show_navigation_info()" in Pages -> management_map.
get the list of elements via "get_navigation_info_elements_list(visible_only=True)" in Pages -> management_map and save it as element_list.
return True if element_name is indeed in element_list.
otherwise return False.

create side function called "verify_events_created(element_ip, expected_events)" that:
go to Alarm & Events page on LW server.
set Faults type to "Events".
set filter by to "Devices" via 'set_filterBy' in 'Pages -> alarms_and_events'
select the ip via 'select_device_filterBy_devices(ip)' in 'Pages -> alarms_and_events'
get the "To date" value via "get_to_date" in 'Pages -> alarms_and_events' and save it as to_date. (format example: 10/05/2026 18:15:44)
take to_date and substract from it 10 minutes (for example: 10/05/2026 18:05:44) and save it as from_date.
set From date via set_from_date(from_date) 'Pages -> alarms_and_events'
get the events from LW Server via 'get_all_events' in 'Pages -> alarms_and_events' and store them as element_events.
make sure expected_events included in element_events by comparing their Message field.
return True if they are included.
otherwise return False.

create side function called "verify_element_is_green(element_name: str | None = None, element_ip: str | None = None, logger)" that:
go to 'Management Map' page.
if element_name:
  get the color of the element via "get_map_element_color(element_name)" in Pages -> management_map.
  you can do like this:
  state, color = management_map.get_map_element_color(element_name=element_name)
  if state != "positive" or color != "green":
     print(f"IP {element_name}: Element color is not green: {color}")
     logger.info(f"IP {element_name}: Element color is not green: {color}")
     return False
if element_ip:
  get the color of the element via "get_map_element_color(element_ip)" in Pages -> management_map.
  you can do like this:
  state, color = management_map.get_map_element_color(element_ip=element_ip)
  if state != "positive" or color != "green":
     print(f"IP {element_ip}: Element color is not green: {color}")
     logger.info(f"IP {element_ip}: Element color is not green: {color}")
     return False
return True
